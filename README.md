# 🌿 PlantCare AI
### Advanced Plant Disease Detection using Transfer Learning

PlantCare AI is a deep learning project that detects plant diseases from leaf images using transfer learning.

## 🚀 Features
- Detect plant diseases from images
- Uses Transfer Learning (MobileNetV2)
- Deep learning image classification
- Trained using TensorFlow/Keras

## 🧠 Model
The project uses **MobileNetV2**, a pretrained convolutional neural network trained on ImageNet.  
The model is fine-tuned to classify plant images.

## 📊 Results
The model achieved approximately **94% accuracy** during training.
