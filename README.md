# PlantCare AI: Advanced Plant Disease Detection using Transfer Learning

This project uses deep learning and transfer learning to detect plant diseases from leaf images.

## Model
The model is built using MobileNetV2 pretrained on ImageNet and fine-tuned for plant classification.

## Technologies Used
- Python
- TensorFlow / Keras
- Transfer Learning
- MobileNetV2

## Dataset
Image dataset used for plant classification.

## Results
The trained model achieved approximately 94% accuracy.
